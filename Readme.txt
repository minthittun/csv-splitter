How to run?

1) Replace "filename.csv" with your CSV file.
2) Change "table_name" variable in program_sql.py.
3) Change row count in "chunk_size". It is located in program_csv.py.
4) open terminal or command prompt.
5) python program_csv.py.
6) python program_sql.py.

Folder structure
CSVs files are located in "csvs" folder.
SQLs filers are located in "sqls" folder.
