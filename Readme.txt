How to run?

1) Change "table_name" variable in program_sql.py
2) Change row count in "chunk_size". It is located in program_csv.py
2) open terminal or command prompt
3) python program_csv.py
4) python program_sql.py

Folder structure
CSVs files are located in "csvs" folder.
SQLs filers are located in "sqls" folder.
